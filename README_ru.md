Это учебный проект, созданный для GitHub.
Он демонстрирует базовые принципы ORM и интеграцию SQLite в Python.

Автор проекта: [Kergochka/Kerga](https://github.com/Kergochka)  
Дата первого релиза: 2026-03-27

Подробности о лицензии смотрите в `LICENSE.md`.

## Структура проекта
- `kerastack/` - основной пакет с реализацией ORM
- `READMI_en` - черновик README на английском
- `READMI_ru` - версия README на русском
- `LICENSE.md` - лицензия проекта
- `ARCHITECTURE.md` - заметки по архитектуре
- `ORM_TEST.py` - файл для тестирования ORM
